What is MCP?

The Model Context Protocol is Anthropic's open standard for connecting AI models to external tools and data sources. Each of the agents above gets a toolbox at session time: the URLs listed here. When an agent needs to look up contact data or check a calendar, it issues a tool call through the MCP server and receives the result. Credentials never live in the agent — they're kept in the vault and injected at session start.
